# ZZ-助手（学习助手智能体）

一个基于 Python 开发的学习助手项目，集成了智能交互、后端服务与用户管理功能。

---

## ✨ 项目功能
- 🤖 智能学习辅助：提供学习任务处理、信息查询与解答能力
- 🔐 用户系统：支持用户注册、登录与数据持久化存储
- 🚀 后端服务：提供 API 接口，可对接前端或其他服务
- 📊 结果记录：自动保存交互结果与测试日志

---

## 🛠️ 技术栈
- 语言：Python 3.12
- 核心模块：`api.py`、`config.py`、`agent.py`
- 环境配置：`.env` 文件管理敏感信息

---

## 📁 项目结构
study_agent/├── .env # 环境变量配置文件
├── api.py # 后端 API 接口
├── config.py # 项目配置文件
├── agent.py # 智能体核心逻辑
├── agent_output.txt # 交互结果输出日志
└── api_test_result.txt # API 测试结果日志
